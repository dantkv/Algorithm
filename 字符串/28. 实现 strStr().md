[28. 实现 strStr()](https://leetcode-cn.com/problems/implement-strstr/) （简单）

#### 解法

暴力枚举，时间复杂度O(m * n)

```C++
class Solution {
public:
    int strStr(string haystack, string needle) {
        int len1 = haystack.length();
        int len2 = needle.length();
        if(len2 == 0) return 0;
        for(int i = 0; i <= len1-len2; i++)
        {
            int j = 0;
            for(;j < len2; j++)
            {
                if(haystack[i+j] != needle[j])
                {
                    break;
                }
            }
            if(j == len2) return i;
        }
        return -1;
    }
};
```

[69. x 的平方根](https://leetcode-cn.com/problems/sqrtx/) （简单）

#### 解法

在0 ~ x 范围使用二分法逼近

```Java
class Solution {
    public int mySqrt(int x) {
        int l = 0;
        int r = x;
        int ans = -1;

        while(l <= r)
        {
            int middle = l + ((r-l)>>1);

            /**
             * 这里运算时是默认整数，运算结果也是是整数，但是结果用整数存储不下，所以需要先转换为long类型
             * 运算时会以更高类型进行运算。
             */
            long sum = (long)middle*middle;
            if(sum <= x)
            {
                ans = middle;
                l = middle+1;
            }
            else
            {
                r = middle-1;
            }
        }
        return ans;
    }
}
```
